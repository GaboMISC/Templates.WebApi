# Instrucciones para Ejecutar el Proyecto

## Pre-requisitos
Asegúrese de tener instalado el runtime de .NET 8.0 en su máquina.

## Ejecución del Proyecto
1. Ejecute los scripts de SQL.
2. Remplace el nombre de su instancia de SQL o la cadena de conexión completa si su servidor tiene usuario y contraseña.

## Notas
Debe de establecer por defecto que proyecto quiere depurar.
